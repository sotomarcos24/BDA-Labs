# Challenge box

## 1. Can you rewrite `my_len` using an index instead of iterating over values?

Answer: Yes. Use `range(len(data))` and increase a counter once for each index.

```python
def my_len(data):
    count = 0
    for i in range(len(data)):
        count += 1
    return count
```

## 2. How would you count only numbers greater than 20 in a list?

Answer: Use a counter and increase it only when the condition is true.

```python
def count_greater_than_20(data):
    count = 0
    for item in data:
        if item > 20:
            count += 1
    return count
```

## 3. What happens if the target value is not in the list? How would you handle it?

Answer: If we do not handle it, the function may print nothing or fail to return a useful result. A good way is to return `-1`.

```python
def find_position(data, target):
    pointer = 0
    for item in data:
        if item == target:
            return pointer
        pointer += 1
    return -1
```

## 4. How would you print only the diagonal values of a matrix?

Answer: Print values where the row index and column index are equal.

```python
matrix = [
    [10, 20],
    [30, 40]
]

for i in range(len(matrix)):
    print(matrix[i][i])
```

## 5. Why is using `break` important when searching? What happens if you remove it?

Answer: `break` stops the loop as soon as the value is found, so it avoids extra work. Without it, the loop keeps going through the rest of the list.

## 7. Can you find the last occurrence of a value instead of the first?

Answer: Yes. Do not use `break`. Keep updating the position whenever the value is found.

```python
def find_last_position(data, target):
    position = -1
    pointer = 0
    for item in data:
        if item == target:
            position = pointer
        pointer += 1
    return position
```

## 8. Can you count elements without using `for` and use `while` only?

Answer: Yes. Use an index that increases until it reaches the end of the list.

```python
def my_len_while(data):
    count = 0
    i = 0
    while i < len(data):
        count += 1
        i += 1
    return count
```

## 9. Can you write a function that counts elements satisfying any condition, such as `> 10` or even?

Answer: Yes. One simple way for beginners is to pass a function as the condition.

```python
def count_by_rule(data, rule):
    count = 0
    for item in data:
        if rule(item):
            count += 1
    return count

print(count_by_rule([1, 2, 3, 4, 5], lambda x: x % 2 == 0))  # 2
```

## 10. When should you use `for item in data` vs `for i in range(len(data))`?

Answer: Use `for item in data` when you only need the values. Use `for i in range(len(data))` when you need the position or want to access items by index.
