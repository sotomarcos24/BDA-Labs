## Welcome to Session 1

This session introduces the core setup and programming concepts used throughout the module.

### Learning goals

By the end of Session 1, you should be able to:

- Set up a Python project with a virtual environment
- Install dependencies from `requirements.txt`
- Use counters, indexes, and nested loops
- Read and process CSV data with Python's built-in `csv` module
- Report basic time/space complexity
