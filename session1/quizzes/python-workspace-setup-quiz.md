# Python basics quiz

## Question 1

Why is Python called “Python”?

- It was named after a type of snake.
- It was named after the comedy group *Monty Python*.
- It was named after a scientist called Python.
- It is an acronym.

Answer: 2
Type: single
Time: 45
Explanation: Python was named after the British comedy group *Monty Python*, not the snake.
