# Big data analytics

Welcome to the Big Data Analytics lab repository. This repo contains all weekly materials including tutorials, exercises, quizzes, cheatsheets, and homework.

## Structure

You can use this repository in two ways:

1. Clone (recommended for most students)
  
Use this if you only want to access materials and work locally.

2. Fork (recommended for reorganizing work)

Create your own copy of the repository on GitHub. This allows you to:
- track your progress
- commit your solutions
- share your work easily

You can also pull updates from the original repository (upstream) when new materials are released.
